# pong
An Atari Pong reboot built by Jeffrey Rehm Feb 25 - March 6 2019

- The game is 0-2 players
- The indicators located at the top corners of each screen are blue when a player is in control and red when a bot is in control of the corresponding paddle

CONTROLS ----------------

UP - move the left paddle up
DOWN - move the left paddle down
W - move the right paddle up
S - move the right paddle down

B - toggle smart bot (left paddle)
N - toggle dumb bot (right paddle)
D - toggle ball target indicator (only when smart bot is active)
I - toggle control indicator visibility
R - reset the ball location and velocity (center)

ESC - close the program
SPACE - pause/start game
ENTER - reset/start game
